# Hangman Game

Hangman is a classic word game. User guesses the letter and eventually the whole word. If the letter is not a fit then they lose life. 

This is a simple project which contains most of the foundation of python, more like programming as a whole. It has conditions - if statements, loops such as for-loop and while-loop, and list with its function and index manipulation. 

The word is small for now as I planned to focus more in logic and game building. I will expand and filter it later as per user choice. Other minor changes will also be done.
