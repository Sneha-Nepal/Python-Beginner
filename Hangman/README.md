# Hangman Game

Hangman is a popular word game. User guesses the letter and eventually the whole word. If the letter is not among the word then they lose life. 

This is a simple project which contains most of the foundation of python, more like programming as a whole. 

## Foundational Concepts:
* **Conditional Statements:** `if-else` statement is used for filtering the word category.
* **Looping:** `for` loop is used to iterate over the placeholder and the string. `while` loop is used to restart the game.
* **List:** `list` are used for index manipulation and for storing the correct letters and eventually the whole word.
* **Import:** Using the `import` keyword, the words from word bank are imported. This helped to not crowd the main program. 

## Uniqueness:
To make it unique I have added a filter feature for the user to choose the word from Science, World, People, Programming, and Random category. Basically, I have categorised the words in `word_bank.py` file.
