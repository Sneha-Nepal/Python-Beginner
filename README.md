# Simple Projects

These are beginner projects to practice foundational skills in Python. It is a collection of common projects, but with unique twists added to them.

### Source
The core project concepts and ideas are inspired by Angela Yu's 100 Days of Code: The Complete Python Pro Bootcamp on Udemy. However, all code was written independently from scratch.

To push my learning journey further, I have added unique features and design improvements to showcase creativity and problem-solving.
