# Caesar-Cipher Program

Caesar-Cipher is a program used for both encrypting (encoding) and decrypting (decoding) text based on a user-specified shift number. 

This project was built independently to master core programming concepts.

## Features

* **Encryption (Encoding):** Shifts letters forward in alphabetical order by a user-defined number.
* **Decryption (Decoding):** Shifts letters backward to restore the original message.
* **Out of Range Error:** Handles large shift numbers seamlessly using mathematical logic.
* **Ignore Punctuations:** Keeps spaces and special characters intact during the process.

## Uniqueness

Unlike normal Cipher programs that only allow a single encryption or decryption, this version features a unique **Double Lock** mode. It gives the user the option to double encrypt or double decrypt their message using two completely different shift numbers for an extra layer of security.

## Concepts Learned & Applied

This project helped solidify foundational Python syntax and clean coding practices, including:
* **Functions:** Used to maximize code reusability and maintain a clean structure.
* **Control Flow & Loops:** `if-else` for conditionals statement, `for` loops to iteration, and `while` loops to restart the game.
* **Lists and Strings:** Manipulating indices, using `join()` to output the final result, and `.index()` to find the indices of letters known.
